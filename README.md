# DID One Identity Platform

> Identity First World for humans, agents, tools, skills, credentials, APIs, devices, organizations, and worlds.

DID One starts with a simple law:

```text
No identity, no action.
No proof, no trust.
No lifecycle, no agent.
No recovery, no production identity.
```

## DIDCore

DIDCore is the identity kernel of DID One World.

```text
DIDCore =
  DID
  + DID Document
  + Controller
  + Verification Method
  + Wallet Binding
  + Credential Anchor
  + Lifecycle State
  + Recovery Policy
  + Compatibility Profile
  + Immutable Audit
```

## Platform doctrine

```text
Identity first.
Proof always.
Trust in chains.
Data in blocks.
Receipts at crossings.
Recover or retire.
Graceful exit.
Backward compatibility.
Open at the end.
```
